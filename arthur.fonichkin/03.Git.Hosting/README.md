## Script:
```bash
CURRENT_BRANCH=`git branch --show-current`
REMOTES=`git remote`

for remote in $REMOTES; do
    echo "Pushing to remote: $remote on branch: $CURRENT_BRANCH"
    git push "$remote" "$CURRENT_BRANCH"
done

echo "Push to all remotes complete!"
```
